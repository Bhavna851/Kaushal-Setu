# Kaushal Setu

## PM Internship Recommendation Portal

Kaushal Setu is a rule-based Internship Recommendation Portal developed using Flask and SQLite. The application recommends internships to students based on their skills, education qualification and preferred location.


## Features

- User Registration and Login
- Student Profile Management
- Rule-based Internship Recommendation
- Skill Matching
- Education Matching
- City and State Matching
- Internship Score Calculation
- Apply for Internship
- Application Tracking
- Real Internship Dataset
- Responsive Dashboard
- Chatbot Support


## Technologies Used

Frontend

- HTML
- CSS
- Bootstrap
- JavaScript

Backend

- Flask
- Python

Database

- SQLite
- SQLAlchemy




## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Kaushal-Setu.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python run.py
```

Open

```
http://127.0.0.1:5000
```
